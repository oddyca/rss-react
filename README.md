# React-TS-Redux App
### Features

 - Fetch from NASA open API
 - Search with the use of the API
 - Form submission with useForm hook
	 - Form validation
 - Loader animation
 - Redux
 - React Router
 - TypeScript

[Netlify](https://nasa-api-form.netlify.app/)
